# pedestrian-behavior-comparison-system
歩行者行動比較システム
# README
##  システム概要
### ２つの条件下で、あるフロアでの歩行者の[滞在]を可視化し、比較するシステム
滞在 = 歩行者が静止する
条件１: 従来
条件 2: お茶会

## 可視化手法
### ヒートマップ
#### 色の強弱は静止時間
滞在時間が長いほど赤くなる 
| 静止時間 | 色       | 
| -------- | -------- | 
| 長い     | 赤🟥   | 
| 短い     | 黄色🟨 | 

### フロアマップ
### 静止の判定方法は歩行速度
| 速さ(m/s)    | 判定 | 
| ------------ | ---- | 
| 0以下        | 静止 | 
| 0 より大きい | 歩行|
